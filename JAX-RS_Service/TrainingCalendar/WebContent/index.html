Training Calendar

JAX-RS service is running....

User services with description

Methods Service url Description
GET http://localhost:8080/TrainingCalendar/rest/tc/user Display all the users
http://localhost:8080/TrainingCalendar/rest/tc/user/searche/{email} Search user with email
http://localhost:8080/TrainingCalendar/rest/tc/user/auth/{email},{password} User login with authentication
POST http://localhost:8080/TrainingCalendar/rest/tc/user Add new user
PUT http://localhost:8080/TrainingCalendar/rest/tc/user Update existing users
DELETE http://localhost:8080/TrainingCalendar/rest/tc/user/{email} Delete a user with email
